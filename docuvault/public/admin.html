👑 Admin Roles Explained
Three permission tiers control what each admin account can do.
👑
Super Admin
Full access to everything, including managing other admin accounts and the colour scheme / branding.
🛠️
Admin
Full access to clients, documents, messages, agreement and email settings — except changing the colour scheme.
📥
Limited Admin
Can only download documents, message clients, and manage document requests (checklist). Cannot create/remove clients, delete documents, or change settings.
